Набор инструментов для ПК 5Bites TK031

Описание товара

Набор инструментов 5Bites TK031

Характеристики товара

  • Материал рукояти — пластик, металл
  • Трещоточный механизм — нет
  • Для точных работ — нет
  • Кейс в комплекте — плотный кофр

Подробное описание товара

Набор инструментов 5Bites TK031 состоит из 44 предметов и предназначен для работы со структурированной кабельной системой. Удобный кофр из плотного материала надежно сохранит содержимое набора внутри. К тому же надежное крепление инструментов по обеим сторонам кейса не позволит им потеряться. Благодаря своим компактным размерам кофр можно брать с собой. Таким образом, инструменты удобны в транспортировке и мобильны в применении. Набор инструментов 5Bites TK031 состоит из ножа для обрезки кабелей, тестера, универсального ножа и клещей для обжима. Набор дополнен коннекторами RJ11 и RJ45 по 20 штук каждого наименования.

Комплектация

  • кабельный тестер, кримпер для обжима
  • нож LY-501B для зачистки и резки кабеля, нож Krone LY-T2021
  • коннекторы RJ11+RJ45 40 шт
